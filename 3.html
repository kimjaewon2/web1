JavaScript

JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web.JavaScript and Java are completely different languages, both in concept and design. JavaScript was invented by Brendan Eich in 1995, and became an ECMA standard in 1997. ECMA-262 is the official name of the standard. ECMAScript is the official name of the language.
